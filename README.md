# PUBGM-ESP-AIMBOT
Pubg Mobile 0.19.0 ESP &amp; AIMBOT Hack using using BypaPH, bypasses process read/write restrictions.

# Tools:
- C#
- VS2019

Features:
- Player ESP (2d box, 3d box,skeleton, ignore teammate, identify bot)
- Item/Lootbox ESP
- Vehicle ESP
- Grenade Alert
- Player Health/Name/Distance/Line
- Simple Aimbot
- Enemy State
- Enemy Weapon
